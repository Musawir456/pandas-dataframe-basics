# Pandas DataFrame 

This repository covers the fundamentals of **Pandas DataFrame**, which is the core data structure used for data analysis and manipulation in Python.

## Topics Covered
- Introduction to DataFrame
- Creating DataFrames
- Viewing DataFrame data
- DataFrame shape and size
- Column and row selection
- Adding and removing columns
- Index handling
- Basic DataFrame operations
- DataFrame overview methods

##  Tools & Libraries
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Purpose
- Understand the structure of Pandas DataFrame
- Build a strong foundation for data analysis
- Prepare for advanced Pandas operations
- Practice DataFrame-based data handling
